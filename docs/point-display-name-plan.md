# 포인트 표시명 설정 계획

이 문서는 포인트 모듈의 운영 표시명을 관리자 환경설정에서 바꿀 수 있게 하기 위한 구현 계획이다.

추적 이슈: [#35 feat: 포인트 표시명 설정 기능 추가](https://github.com/whitedot/saanraan/issues/35)

문서 수명:

- 포인트 표시명 설정 기능을 구현하기 전까지 계획 문서로 보관한다.
- 실제 구현과 검증이 끝나면 이 문서는 삭제한다.
- 구현 후 계속 유지해야 하는 기준은 `docs/module-guide.md`, `docs/core-decisions.md`, 포인트 모듈 README 또는 Wiki 관리자 화면 설명으로 옮긴다.

## 기본 방향

포인트 모듈에 독립 환경설정 화면을 추가한다.

권장 화면:

- `GET /admin/points/settings`: 포인트 환경설정 화면
- `POST /admin/points/settings`: 포인트 환경설정 저장

관리자 메뉴는 포인트 모듈 아래에 `환경설정`, `잔액`, `거래 내역` 순서로 둔다. 기존 `/admin/points` 진입은 환경설정 또는 잔액 중 구현 시점의 운영 흐름에 맞춰 redirect하되, 메뉴에서 설정 화면에 직접 접근할 수 있어야 한다.

## 설정 항목

최소 설정:

- 표시명: 기본값 `포인트`
- 단위명: 기본값 `P` 또는 `포인트`

저장 위치:

- 우선 `sr_module_settings`에 `module_key=point` 범위로 저장한다.
- 포인트 모듈 전용 테이블은 표시명만으로는 만들지 않는다.

검증 기준:

- 표시명은 공백 제거 후 비어 있으면 저장을 거부한다.
- 표시명과 단위명은 운영 화면에 출력되므로 길이를 제한하고 출력 시 항상 escape한다.
- 서버 검증을 기준으로 하고, 화면에는 `(필수)` 표기와 `required` 보조 검증을 함께 둔다.

## 표시명 helper

포인트 모듈은 다른 모듈이 직접 설정 key를 알지 않아도 되도록 좁은 helper를 제공한다.

예상 helper:

```php
sr_point_display_name(PDO $pdo): string
sr_point_unit_label(PDO $pdo): string
sr_point_asset_option(PDO $pdo): array
```

`sr_point_asset_option()`은 콘텐츠/커뮤니티 같은 자산 소비 모듈에서 선택지 라벨을 만들 때 사용할 수 있는 형태로 반환한다.

## 다른 모듈 적용 범위

콘텐츠와 커뮤니티처럼 회원 자산을 선택하거나 표시하는 모듈은 포인트 자산 라벨을 하드코딩하지 않는다.

적용 대상:

- 콘텐츠 유료 열람 자산 선택
- 콘텐츠 파일 다운로드 과금 자산 선택
- 콘텐츠 완료 버튼 자산 처리 선택
- 커뮤니티 글쓰기/댓글 차감 자산 선택
- 커뮤니티 게시글 유료 열람 자산 선택
- 커뮤니티 첨부 다운로드 차감 자산 선택
- 관리자 안내 문구의 차감 우선순위 표시

동작 기준:

- 포인트 모듈에서 표시명을 `마일리지`로 설정하면 다른 모듈의 자산 선택 UI와 우선순위 문구에서도 `포인트` 대신 `마일리지`가 보인다.
- 포인트 모듈이 비활성화되어 있거나 helper를 사용할 수 없으면 기존 기본 표시명 `포인트`로 fallback한다.
- 적립금, 예치금은 각 모듈의 현재 표시 정책을 유지한다. 같은 요구가 생기면 각 자산 모듈의 설정으로 확장한다.

## 제외 범위

이번 계획은 표시명과 단위명만 다룬다.

제외:

- 포인트 원장 테이블명 또는 거래 참조명 변경
- 포인트 계산 정책 변경
- 포인트 만료 정책 추가
- 적립금/예치금 표시명 설정
- 기존 거래 데이터 마이그레이션

## 검증

수동 확인:

- `/admin/points/settings`에서 표시명과 단위명을 저장할 수 있다.
- 빈 표시명 저장은 서버에서 실패한다.
- `/admin/points/balances`, `/admin/points/transactions`, `/account/points`에서 새 표시명이 보인다.
- 콘텐츠 관리자 자산 선택 UI에서 포인트 대신 설정 표시명이 보인다.
- 커뮤니티 관리자 자산 선택 UI에서 포인트 대신 설정 표시명이 보인다.
- 차감 우선순위 안내 문구가 새 표시명을 사용한다.
- 포인트 모듈 비활성 또는 설정 누락 상황에서는 기본 표시명으로 깨지지 않는다.
