# 운영자 대상 기능 소개

이 문서는 일반 사용자 또는 사이트 운영자에게 `saanraan`의 현재 구현 기능을 소개할 때 사용할 수 있는 홍보용 기능 목록이다. 실제 저장소에 포함된 본체와 번들 모듈을 기준으로 작성하며, 본인확인, 결제, 회원 마이그레이션처럼 계획 단계에 있는 기능은 구현 기능으로 설명하지 않는다.

`saanraan`은 저가형 PHP 웹호스팅에서도 운영할 수 있는 회원 중심 모듈형 웹 솔루션 베이스다. 작은 코어 위에 회원, 콘텐츠, 커뮤니티, 포인트/금액, 쿠폰, 알림, 개인정보, 사이트 운영 기능을 모듈 단위로 올려 필요한 기능부터 선택해 확장할 수 있다.

## 한눈에 보는 운영 포인트

| 기능 | 안내 문구 | 주요 기능 |
| --- | --- | --- |
| 모듈형 운영 | 사이트 성격에 맞춰 필요한 기능을 설치하고 활성화하는 방식으로 운영할 수 있다. | 회원, 관리자, 개인정보, 콘텐츠, 커뮤니티, 사이트 운영, 자산 모듈 번들 제공 |
| 공유호스팅 친화 구조 | 복잡한 서버 구성보다 일반 PHP 호스팅에 올리기 쉬운 구조를 지향한다. | PHP 8.1 이상, MySQL 또는 MySQL 호환 DB, Vanilla JavaScript, plain CSS |
| 회원 중심 확장 | 회원 계정, 그룹, 자산, 쿠폰, 알림, 개인정보 요청을 한 흐름 안에서 연결한다. | 회원 관리, 회원 그룹, 포인트/적립금/예치금, 쿠폰·이용권, 알림, 개인정보 요청 |
| 운영 안전성 | 관리자 권한과 기록, 보안 helper를 통해 운영 중 확인해야 할 흔적을 남긴다. | 관리자 권한, 감사 로그, 보존 정책, CSRF, 출력 escape, rate limit |
| 작은 코어 | 코어는 실행 기반에 집중하고 콘텐츠, 커뮤니티, 쿠폰 같은 도메인 정책은 모듈이 소유한다. | 설치/업데이트, DB 연결, 설정, 모듈 조회, 보안 helper, 출력 슬롯 |

## 설치와 기본 운영

| 기능 | 안내 문구 | 주요 기능 |
| --- | --- | --- |
| 웹 설치 | 처음 설치하는 운영자가 복잡한 CLI 작업 없이 설치 흐름에 진입할 수 있다. | 미설치 상태 자동 설치 흐름, DB 기반 사이트 설정 저장 |
| 모듈 설치와 활성화 | 운영 중 필요한 기능을 모듈 단위로 켜고 관리할 수 있다. | 모듈 설치, 활성화, 비활성화, 모듈 상태 관리 |
| 업데이트 관리 | 본체와 모듈의 schema version을 기준으로 업데이트 상태를 확인한다. | 본체 schema version, 모듈 schema version, update SQL 구조 |
| 배포 친화 구성 | 저가형 공유호스팅에서도 부담이 적은 PHP 중심 배포 구조를 사용한다. | PHP 파일 기반 배포, Apache 호환 보호 기준, `sr_` DB prefix |
| 저장소 기반 | 파일 저장과 외부 저장소 연동을 위한 기본 helper를 제공한다. | 로컬 파일 저장 helper, S3 호환 저장소 helper |

| 보존 정책 | 운영 데이터 보존 기간을 별도 화면에서 관리할 수 있다. | 운영 데이터 보존 정책 관리 |
| 관리자 UI | 반복되는 관리자 화면을 일관된 구성으로 만들기 위한 UI 기준을 제공한다. | 관리자 UI kit, 공통 UI 컴포넌트, 목록 필터, 검색, 페이지네이션 helper |
| 요청 보호 | 관리자 저장 작업은 권한과 CSRF 검증을 거쳐 처리한다. | 관리자 POST action CSRF 검증, 로그인 검사 |

## 회원 관리

| 기능 | 안내 문구 | 주요 기능 |
| --- | --- | --- |
| 회원가입과 로그인 | 회원 기반 사이트 운영에 필요한 기본 인증 흐름을 제공한다. | 회원가입, 로그인, 로그아웃, 이메일 기반 계정, 로그인 ID |
| 이메일 인증 | 회원 이메일 신뢰도를 확인하고 인증 상태를 관리할 수 있다. | 이메일 인증, 이메일 인증 여부 관리 |
| 비밀번호 관리 | 회원이 비밀번호를 잊었거나 변경해야 할 때 사용할 수 있는 기본 흐름을 제공한다. | 비밀번호 재설정, 비밀번호 변경 |
| 내 계정 | 회원이 자신의 기본 정보를 확인하고 수정할 수 있다. | 회원 프로필 관리, 회원 정보 수정, 회원 탈퇴 |
| 세션과 인증 로그 | 로그인 상태와 인증 이력을 기록해 계정 운영 상황을 확인할 수 있다. | 회원 세션 관리, 로그인 및 인증 로그 기록 |
| 관리자 회원 관리 | 운영자가 회원을 검색하고 상태와 기본 정보를 관리할 수 있다. | 관리자 회원 목록, 회원 생성, 회원 정보 수정, 회원 상태 관리, 회원 검색과 필터 |
| 회원 식별 | 내부 ID 노출을 줄이기 위해 공개 해시 기반 식별을 지원한다. | 회원 공개 해시 기반 식별 |
| 회원 그룹 | 회원을 그룹으로 나누고 접근 정책이나 운영 정책과 연결할 수 있다. | 회원 그룹 관리, 수동 배정과 해제, 회원 그룹 규칙 관리, 회원 그룹 평가 화면 |
| 회원 메뉴 연동 | 회원이 자신과 관련된 운영 기능을 계정 영역에서 확인할 수 있다. | 포인트, 적립금, 예치금, 쿠폰, 알림, 개인정보 요청 메뉴 연동 |

| 변경 기록 | 콘텐츠 변경 작업을 감사 로그로 남겨 운영 이력을 확인할 수 있다. | 콘텐츠 변경 감사 로그 |

## 현재 단계 안내 문구

외부 소개나 운영자 안내에는 다음 문구를 함께 사용하는 것을 권장한다.

> `saanraan`은 현재 1.0 릴리스 전 안정화 단계의 모듈형 PHP 웹 솔루션 베이스입니다. 회원, 관리자, 콘텐츠, 커뮤니티, 포인트/금액, 쿠폰, 알림, 개인정보, 사이트 운영 모듈이 구현되어 있으며, 본인확인, 결제, 회원 마이그레이션 기능은 별도 계획 단계입니다.
