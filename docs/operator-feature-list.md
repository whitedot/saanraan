# 운영자 대상 기능 소개

이 문서는 일반 사용자 또는 사이트 운영자에게 `saanraan`의 현재 구현 기능을 소개할 때 사용할 수 있는 홍보용 기능 목록이다. 실제 저장소에 포함된 본체와 번들 모듈을 기준으로 작성하며, 본인확인, 결제, 회원 마이그레이션처럼 계획 단계에 있는 기능은 구현 기능으로 설명하지 않는다.

`saanraan`은 저가형 PHP 웹호스팅에서도 운영할 수 있는 회원 중심 모듈형 웹 솔루션 베이스다. 작은 코어 위에 회원, 콘텐츠, 커뮤니티, 포인트/금액, 쿠폰, 알림, 개인정보, 사이트 운영 기능을 모듈 단위로 올려 필요한 기능부터 선택해 확장할 수 있다.

## 한눈에 보는 운영 포인트

운영자는 관리자 화면에서 사이트 기본 설정, 회원, 콘텐츠, 커뮤니티, 자산, 쿠폰, 알림, 개인정보 요청을 한 흐름 안에서 다룰 수 있다. 각 기능은 모듈로 분리되어 있어 사이트 성격에 맞춰 필요한 기능을 설치하고 활성화하는 방향으로 운영할 수 있다.

- 공유호스팅 친화적인 PHP 8.1 이상, MySQL 또는 MySQL 호환 DB 기반
- 웹 설치와 설치 후 모듈 설치, 활성화, 업데이트 흐름 제공
- 회원, 관리자, 개인정보, 콘텐츠, 커뮤니티, 사이트 운영, 자산 모듈 번들 포함
- 포인트, 적립금, 예치금, 쿠폰을 콘텐츠/커뮤니티 유료 접근과 연결
- 관리자 권한, 감사 로그, 보존 정책, 보안 helper 기반 제공
- 코어는 작게 유지하고 도메인 기능은 모듈별로 소유

## 설치와 기본 운영

처음 설치하는 운영자가 복잡한 서버 구성을 먼저 배워야 하지 않도록, 설치와 업데이트 흐름을 웹 기반으로 제공한다. 운영 중에는 관리자 화면에서 모듈 상태와 업데이트 상태를 확인하며 사이트 기능을 점진적으로 확장할 수 있다.

- 미설치 상태에서 웹 설치 흐름으로 진입
- DB 기반 사이트 설정 저장
- 기본 DB prefix `sr_` 사용
- 모듈별 설치 SQL과 update SQL 구조 제공
- 관리자 화면에서 모듈 설치, 활성화, 비활성화, 업데이트 관리
- 본체 및 모듈 schema version 관리
- 로컬 파일 저장 helper 제공
- S3 호환 저장소 helper 제공
- Apache 또는 Apache 호환 공유호스팅 배포 보호 기준 제공
- PHP, Vanilla JavaScript, plain CSS 중심의 단순한 배포 구조

## 관리자 대시보드와 운영 관리

관리자는 로그인 후 대시보드에서 사이트 운영 상태를 확인하고, 주요 모듈의 관리 화면으로 이동할 수 있다. 관리자 기능은 사이트 운영에 필요한 공통 흐름을 제공하되, 콘텐츠나 커뮤니티 같은 도메인 관리는 각 모듈 화면에서 담당한다.

- 관리자 대시보드
- 모듈별 대시보드 요약 섹션
- 사이트 설정 관리
- 홈페이지 설정 관리
- 관리자 메뉴 관리
- 관리자 역할 관리
- 관리자 권한 관리
- 관리자 감사 로그 조회
- 운영 데이터 보존 정책 관리
- 업데이트 화면
- 관리자 UI kit 및 공통 UI 컴포넌트
- 관리자 목록 필터, 검색, 페이지네이션 helper
- 관리자 POST action CSRF 검증
- 관리자 화면별 권한 검사

## 콘텐츠 관리

운영자는 단일 콘텐츠와 콘텐츠 그룹을 만들고 공개 URL로 노출할 수 있다. 단순 게시를 넘어 유료 열람, 다운로드 과금, 완료 버튼 처리까지 연결되어 강의, 자료실, 멤버십 콘텐츠 같은 운영 흐름의 기반으로 사용할 수 있다.

- 콘텐츠 작성, 수정, 숨김 처리
- 콘텐츠 상태 관리: `draft`, `published`, `hidden`
- `/content/{slug}` 공개 URL
- 콘텐츠 slug 중복 및 예약어 차단
- 관리자 draft 미리보기
- 콘텐츠 제목, 요약, 본문 관리
- SEO 제목과 설명 설정
- plain text 본문 저장과 escape 출력
- CKEditor 연동 시 HTML 본문 저장 지원
- 서버 측 HTML sanitizer 적용
- 콘텐츠 그룹 생성, 수정, 관리
- `/content/group?key={group_key}` 콘텐츠 그룹 공개 목록
- 콘텐츠와 콘텐츠 그룹을 사이트 메뉴 후보로 제공
- sitemap 후보 제공
- 콘텐츠별 공용 배너 선택
- 콘텐츠별 공용 팝업레이어 선택
- 콘텐츠 출력 위치 확장 슬롯 제공
- 콘텐츠 변경 감사 로그

## 콘텐츠 유료 열람과 파일 다운로드

콘텐츠 모듈은 회원 자산 모듈과 연결되어 유료 열람과 파일 다운로드 과금을 처리한다. 운영자는 무료 콘텐츠와 유료 콘텐츠를 함께 운영하고, 최초 1회 차감 또는 반복 차감 같은 정책을 선택할 수 있다.

- 포인트, 적립금, 예치금 기반 유료 열람
- 여러 자산을 함께 쓰는 복합 차감
- 최초 1회 차감 정책
- 매 열람 차감 정책
- 로그인 회원 대상 유료 접근 확인
- 콘텐츠 접근권 기록
- 콘텐츠 다운로드 파일 업로드
- 파일별 다운로드 과금
- 다운로드 내역 조회
- 다운로드 최초 1회 또는 매 다운로드 차감 정책
- 완료 버튼 기반 포인트/금액 지급 또는 차감
- 회원별 완료 버튼 1회 처리
- 유료 열람, 다운로드, 완료 버튼 처리 로그 기록
- 개인정보 사본 제공 export 연동
- 회원 탈퇴/익명화 cleanup 연동

- 게시글 신고
- 관리자 신고 관리
- 스크랩

- 관리자 댓글 관리
- 관리자 게시판별 설정

회원 자산 기능은 포인트, 적립금, 예치금을 각각 독립 원장으로 관리한다. 운영자는 잔액과 거래 내역을 확인하고, 필요한 경우 사유와 참조 정보를 남겨 지급, 차감, 환불을 처리할 수 있다.

- 회원 적립금 잔액과 거래 내역 조회
- 회원 예치금 잔액과 거래 내역 조회
- 관리자 포인트 잔액 목록
- 관리자 적립금 잔액 목록
- 관리자 예치금 잔액 목록

- 자산별 1회 조정 상한 검증

- 개인정보 export 연동

## 포인트/금액 환전

포인트, 적립금, 예치금처럼 성격이 다른 회원 자산을 운영자가 정한 정책에 따라 환전할 수 있다. 환전 정책과 실행 로그가 분리되어 있어 운영자는 허용된 환전 흐름을 관리하고 사후 기록을 확인할 수 있다.

- 포인트, 적립금, 예치금 간 환전 정책 관리
- 환전 가능 자산 설정
- 환전 비율과 수수료 정책 기반 처리
- 회원 환전 화면
- 회원 환전 실행
- 관리자 환전 로그 조회
- 환전 환경설정
- 무수수료 양방향 정책의 순환 가치 증가 방지
- 환전 처리 중 DB 잠금 오류 재시도
- 완료 환전 묶음 정정 helper 제공

## 쿠폰과 이용권

쿠폰·이용권 모듈은 콘텐츠와 커뮤니티 접근권을 회원에게 지급하고 사용 이력을 관리하는 기능을 제공한다. 유료 콘텐츠나 유료 게시판을 운영할 때 쿠폰을 먼저 적용하고, 쿠폰이 없으면 포인트/적립금/예치금 차감으로 이어지게 구성할 수 있다.

- 쿠폰 지급

- 커뮤니티 게시글 열람권 연동
- 유료 콘텐츠/커뮤니티 접근 시 쿠폰 우선 사용

- 사이트 메뉴 관리
- 메뉴 항목 관리
- 헤더 내비게이션 출력
- 푸터 내비게이션 출력
- 콘텐츠 항목 메뉴 후보 연동
- 콘텐츠 그룹 메뉴 후보 연동
- 커뮤니티 게시판 메뉴 후보 연동
- 커뮤니티 게시판 그룹 메뉴 후보 연동
- 활성 메뉴와 활성 메뉴 항목 대시보드 요약
- 사이트 공통 레이아웃 출력 슬롯 연동

- 기본 로고 관리
- 로고 이미지 업로드
- 로고 제목 관리
- alt text 관리
- 링크 URL 관리
- 사용 위치 key 관리
- 이벤트 로고 등록
- 이벤트 로고 노출 기간 설정
- 이벤트 로고 정렬 순서 설정
- 기본 로고와 기간별 로고 전환 기반 제공

## 배너

배너 모듈은 콘텐츠와 커뮤니티의 출력 위치에 맞춰 홍보 영역을 구성할 수 있게 해준다. 운영자는 배너 상태와 노출 대상을 관리하고, 클릭 action을 통해 배너 반응 기록을 남길 수 있다.

- 배너 목록 관리
- 배너 생성, 수정, 삭제
- 배너 상태 관리
- 배너 설정 관리
- 출력 슬롯 기반 배너 노출
- 콘텐츠 출력 위치 대상 배너 노출
- 커뮤니티 출력 위치 대상 배너 노출
- 배너 이미지 출력 action
- 배너 클릭 처리
- 활성 배너와 임시저장 배너 대시보드 요약

## 팝업레이어

팝업레이어 모듈은 공지, 이벤트, 안내 메시지를 사이트 화면에 띄우는 운영 기능이다. 출력 위치와 상태를 관리할 수 있어 특정 콘텐츠나 커뮤니티 화면에 맞춘 안내를 노출하는 데 사용할 수 있다.

- 팝업레이어 목록 관리
- 팝업레이어 생성, 수정, 삭제
- 팝업레이어 상태 관리
- 팝업레이어 설정 관리
- 출력 위치 기반 팝업 노출
- public CSS와 JavaScript 제공
- 활성 팝업과 임시저장 팝업 대시보드 요약

## SEO

SEO 모듈은 사이트가 검색엔진에 기본 정보를 제공할 수 있도록 robots와 sitemap 출력을 담당한다. 콘텐츠와 커뮤니티가 제공하는 sitemap 후보를 모아 검색 노출을 보조한다.

- SEO 설정 관리
- `robots.txt` 출력
- `sitemap.xml` 출력
- 콘텐츠 sitemap 후보 연동
- 커뮤니티 sitemap 후보 연동
- SEO 이미지 출력 action
- 모듈별 SEO 정보와 공통 출력 흐름 분리

## 알림

알림 모듈은 사이트 알림과 이메일 발송 대기열을 관리한다. 운영자는 회원에게 전달되는 알림과 발송 작업 상태를 확인하고, 도메인 이벤트 템플릿을 통해 반복 알림 흐름의 기반을 마련할 수 있다.

- 사이트 알림 생성
- 관리자 알림 목록
- 회원 알림 목록
- 알림 읽음 처리
- 이메일 delivery queue 관리
- 알림 발송 작업 상태 조회
- 알림 이벤트 템플릿 기반 제공
- 알림 설정 관리
- 대시보드 알림 요약
- 개인정보 export 연동

## 개인정보 요청

개인정보 모듈은 회원의 개인정보 요청과 사본 제공 흐름을 조정한다. 각 모듈이 자신의 개인정보 export와 cleanup 계약을 제공하고, 개인정보 모듈은 운영자가 요청을 확인하고 처리할 수 있는 화면을 제공한다.

- 회원 개인정보 요청 화면
- 관리자 개인정보 요청 관리
- 개인정보 사본 제공 요청 처리
- 모듈별 `privacy-export.php` 계약 연동
- 모듈별 cleanup 흐름 연동
- 회원 탈퇴와 익명화 흐름 연결
- 개인정보 요청 export action 제공
- 개인정보 처리 운영 화면 제공

## CKEditor 플러그인

CKEditor 플러그인은 필요한 textarea에 에디터를 붙여 콘텐츠와 커뮤니티 본문 작성 경험을 개선한다. 에디터 로딩에 실패하거나 플러그인을 사용하지 않는 경우에도 기존 textarea 제출 흐름을 유지한다.

- CKEditor 5 선택 플러그인 제공
- 관리자 설정 화면
- 직접 호스팅 방식 지원
- CDN 방식 설정 지원
- CKEditor 에셋 로딩
- textarea 기반 에디터 강화
- 에디터 로딩 성공 시 HTML 본문 저장
- 에디터 비활성 또는 로딩 실패 시 plain textarea 저장 유지
- 콘텐츠/커뮤니티 등 화면 소유 모듈이 저장 정책 소유
- 서버 측 HTML sanitizer와 함께 사용 가능

## 보안과 운영 안정성

운영자가 안심하고 기능을 켤 수 있도록, 관리자 action과 공개 action에는 권한, CSRF, 출력 escape, rate limit 같은 기본 보안 기준을 둔다. 또한 감사 로그와 개인정보 처리 계약을 통해 운영 중 확인해야 할 기록을 남기는 방향을 지향한다.

- 관리자 로그인과 권한 검사
- 관리자 POST CSRF 검증
- 출력 escape helper
- rate limit helper
- 세션 관리
- 토큰 원문이 아닌 토큰 hash 저장 기준
- 관리자 감사 로그
- 회원 인증 로그
- 개인정보 export 계약
- 개인정보 cleanup 계약
- DB 접근 정책 문서
- 보안 모델 문서
- 보안 체크리스트 문서
- 배포 보호 기준 문서

## 현재 단계 안내 문구

외부 소개나 운영자 안내에는 다음 문구를 함께 사용하는 것을 권장한다.

> `saanraan`은 현재 1.0 릴리스 전 안정화 단계의 모듈형 PHP 웹 솔루션 베이스입니다. 회원, 관리자, 콘텐츠, 커뮤니티, 포인트/금액, 쿠폰, 알림, 개인정보, 사이트 운영 모듈이 구현되어 있으며, 본인확인, 결제, 회원 마이그레이션 기능은 별도 계획 단계입니다.

## 홍보 문구 예시

`saanraan`은 작은 PHP 코드베이스 위에 회원, 콘텐츠, 커뮤니티, 포인트/금액, 쿠폰, 알림, 개인정보 기능을 모듈식으로 쌓아 올린 웹 솔루션 베이스입니다. 공유호스팅에서도 운영 가능한 단순한 배포 구조를 지향하면서, 운영자가 실제로 필요한 관리자 화면과 회원 중심 기능을 함께 제공합니다.

콘텐츠를 공개하고, 커뮤니티를 운영하고, 포인트나 쿠폰으로 유료 접근을 구성하고, 개인정보 요청까지 한 흐름 안에서 다루고 싶은 소규모 회원 사이트에 잘 맞습니다.
