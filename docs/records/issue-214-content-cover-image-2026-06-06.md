# Issue 214 Content Cover Image

## 결정

- 콘텐츠 대표 이미지는 콘텐츠 모듈이 `sr_content_items.cover_image_url`로 소유한다.
- 관리자 콘텐츠 생성/수정 화면은 URL 입력과 이미지 파일 업로드를 모두 제공한다.
- 업로드 파일은 jpg, png, webp만 허용하고 재인코딩 후 `content/cover-images/{Y}/{m}` 저장소 키로 저장한다.
- 공개 URL이 직접 제공되지 않는 저장소는 `/content/cover-image?file=...` 프록시로 출력한다.
- 커버 이미지가 없으면 공개 홈과 그룹 목록에서 회색 플레이스홀더를 출력한다.

## 반영 범위

- 콘텐츠 홈 최신 1건 히어로와 최근 콘텐츠 목록에서 커버 이미지를 사용한다.
- 콘텐츠 그룹 목록 썸네일과 콘텐츠 상세 상단 커버에 같은 값을 사용한다.
- 콘텐츠 상세 OG 이미지에도 커버 이미지 URL을 연결한다.
