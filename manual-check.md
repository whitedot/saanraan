# 수동 화면 점검 체크리스트

이 문서는 브라우저에서 직접 화면을 확인하면서 이상하거나 어색한 부분을 기록하기 위한 목록이다. 자동 점검이나 기능 수정은 하지 않고, 화면 상태를 눈으로 확인하는 데 초점을 둔다.

## 공통 확인 기준

- [ ] 첫 화면 진입 시 PHP 오류, warning, notice가 보이지 않는다.
- [ ] 페이지 제목, 본문 문구, 버튼 문구가 어색하지 않다.
- [ ] 폼 라벨, 도움말, 오류 메시지가 실제 입력 항목과 맞다.
- [ ] 버튼, 링크, 탭, 필터, 페이지네이션의 간격과 정렬이 자연스럽다.
- [ ] 모바일 폭에서 텍스트가 겹치거나 잘리지 않는다.
- [ ] 데스크톱 폭에서 표, 카드, 폼이 지나치게 벌어지거나 깨지지 않는다.
- [ ] 성공 / 실패 / 권한 없음 / 빈 목록 상태가 이해 가능하게 표시된다.
- [ ] 로그인 필요 화면은 로그인 후 요청 직전 내부 화면으로 돌아가고, 돌아갈 수 없으면 첫 화면으로 이동한다.
- [ ] 관리자 화면은 권한 없는 사용자에게 노출되지 않는다.
- [ ] 저장, 삭제, 적용 같은 위험 동작은 의도와 결과가 명확하다.
